## 项目说明

目前项目有两个分支

- `Desktop`分支基于`vite + electron`搭建的桌面端应用程序

- 如果你是一个开发者, 你可以选择`Node`分支, 该分支代码提供ICNS解析规则及ICNS转PNG, ICO用例。

## 项目简介

项目主要做了如下内容:

- [x] ICNS解析

- [x] ICNS转PNG

- [x] ICNS转ICO
